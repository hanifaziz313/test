Table

jika kita ingin membuat table pada bootstrap yaitu cukup memberi class "table" pada tag TABLE,dan jika ingin memberi background pada table bisa di per baris ,satu table,atau perkolom saja. bisa juga memberi variaasi belang cukup memberi class "table-striped" atau jika ingin memberi hover maka cukup membri class "table-hover" sebagai mana yang telah teraplikasi pada table di bawah ini dan masih bnyak style yang bisa di implementasikan tinggal kita lihat di bootsrap TABLE.
kita bisa memberi garis pada table yg bisa kita ambil di bootstrap,dan bisa juga memberi warna garis table tsb. kita bisa memberi ukuran kecil pada table dg class "table-sm",dan kita bisa memberi responsive pada table dengan hanya memberi class "table-responsive" pada div pembungkus table,maka dengan otomatis table kita menyesuaikan di semua display layar.

POPOVER

Yaitu button yang menyimpan informasi yang apa bila kita klik akan muncul,kita bisa ambil Popover di bootsrap,dan kita perlu menambahkan tag script yg di dalamya "var popoverTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]')) var popoverList = popoverTriggerList.map(function (popoverTriggerEl) { return new bootstrap.Popover(popoverTriggerEl) })" supaya POPOVER kita berjalan dan perlu di ingat "data-bs-toggle="popover"harus sama antara di bootsrap dan di scripnya,"data-bs-placement="top" " jika kita ingin menggati munculnya informasi pada popover yaitu dengan menggati "top " dengan right,bottom,left,kita juga bisa memberi title pada isi popover "title="aziz" ". Bisa juga jika kita ingin menutup POPOVER denga klik di bagiana mana saja dengan tanpa harus klik di buttonnya yaitu dengan cara tag "button" di ganti dengan tag "a" dan menambahkan "tabindex="0" dan "data-bs-trigger="focus".
